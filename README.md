a. Project Title & Description

SleepZone

SleepZone is a sleep tracking web application designed to help users monitor, analyse, and improve their sleep habits. It allows users to log sleep data, view sleep history, analyse insights, and receive personalised sleep recommendations based on age. The system also includes a sleep calculator, sleep ambience section, and sleep tips to support better sleep health and lifestyle improvement.

b. Features Included

1)Login / Index Page

* Entry point of the system
* Users log in using credentials
* New users are redirected to create profile page
* Existing users are redirected to the dashboard

2)Create Profile Page

* Allows user to create profile before accessing system
* User can select avatar
* User can enter name and age
* Profile data is stored using localStorage

3) Dashboard Page

* Displays sleep summary statistics
* Visual charts including:

  * Average sleep duration
  * Sleep quality score
  * Weekly sleep duration chart
  * Average sleep duration is calculated based on recorded sleep logs
  * Charts will not appear if no sleep records have been added yet

* Sleep ambience section with embedded YouTube videos for relaxation
* Sleep tips section to improve sleep habits and quality

4) Sleep Log Page

* Allows users to input daily sleep data
* Stores sleep time, wake time, and related sleep information

5) Sleep History Page

* Displays all recorded sleep logs in a structured table format
* Includes sleep insights generated from user data such as:

  * Sleep duration analysis
  * Sleep quality interpretation
  * Behavioural patterns

6) Sleep Calculator Page

* Calculates recommended sleep duration based on user age
* Suggests bedtime or wake-up time depending on selected mode

7) About Page

* Provides information about SleepZone system
* Explains purpose of the application and its features
* Describes how it helps users improve sleep habits

8) Profile Page

* Displays user profile information
* Allows user to update name and age
* Shows selected avatar
